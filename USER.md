# Choome User Guide (Quick Start)

## Run the app (technical users)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app in development:
   ```bash
   npm run start
   ```

## Run the app without a terminal (packaged)
1. Build an installer:
   ```bash
   npm run make
   ```
2. Find the installer in `out/make/` and run it.

## Tips
- Recordings are saved to your Videos folder by default.
- You can change the storage folder in Settings.
